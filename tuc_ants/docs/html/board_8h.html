Determines ability to jump.

Author
Intelligence TUC.
Parameters
[in]rowThe row
[in]colThe col
[in]playerThe player
posThe position
Note
It can also be used to determine if we can make a jump from a position we do not occupy. This would happen by providing the proper [row, col] tuple of the desired cell.
Returns
One of following values (0, 1, 2 or 3): 
            - 1 if left jump can be done. 
            - 2 if right jump can be done. 
            - 3 if both directions can be jumped. 
            - 0 if no jump is possible.
Warning
Does not check if we are inside the board
